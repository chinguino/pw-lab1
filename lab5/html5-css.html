HTML5 e CSS

HTML5

HTML5 é a última evolução no padrão que define o HTML. O termo representa dois conceitos diferentes. Esta é uma nova versão da linguagem HTML, com novos elementos, atributos e comportamentos, e um conjunto maior de tecnologias que permite a criação de sites e aplicações da Web mais diversos e poderosos. Este conjunto às vezes é chamado de HTML5 e amigos e, muitas vezes, abreviado apenas para HTML5.

Elementos Usei ou não? Onde utilizei?
header indica cabeçalho (normalmente com logotipos).
nav indica menu de navegação com links para as páginas (só deve haver um por página)
main indica conteúdo principal e único, que não se repete.
section especifica uma parte do conteúdo, usada em conjunto com etiquetas h1|h2|…
footer indica rodapé da secção (com copyright).
arcticle indica porção do documento que pode ser vista de forma independente
aside indica anotações, publicidade ou outros assuntos desconexos.
details define detalhes adicionais que o utilizador pode ver ou esconder, contidos num elemento summary
meter define uma medida escalar estática dentro de um intervalo
progress define uma barra de progresso de uma tarefa
abbr indica o significado de uma abreviatura em conjunto com o atributo title
address informação sobre o contacto do autor

CSS

É um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web

Elementos Usei ou não? Onde utilizei?
div p div p – todos os p que estão “dentro” de um div
div > p div > p – todos os p que estão “logo dentro” de div
div ~ p div ~ p – todos os p que estão colocados depois de um div
div + p div + p – todos os p que estão colocados “imediatamente” depois de um div
grid-template-rows No website todo
grid-template-columns No website todo
grid-template-areas No website todo
@keyframes Na galeria de fotos (todas as fotos têm animações) e na página de introdução
